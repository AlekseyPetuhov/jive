API documentation

package org.aswing.dnd

DefaultDragImage

The default drag image.

DirectlyRemoveMotion

Remove the dragging movieclip directly.

DragListener

Drag and Drop listener.

DragManager

Drag and Drop Manager.

Thanks Bill Lee for the original DnD implementation for AsWing AS2 version.

DraggingImage

The Image for dragging representing.

DropMotion

The motion when a drag action dropped. The motion must remove the drag movie clip when motion is completed.

RejectedMotion

The motion of the drop target does not accept the dropped initiator.

SourceData

The drag source data.
