class org.aswing.event.FocusKeyEvent extends KeyboardEvent

Class Fields

static var FOCUS_KEY_DOWN:String

The FocusKeyEvent.FOCUS_KEY_DOWN constant defines the value of the type property of the event object for a focusKeyDown event.

The properties of the event object have the following values:

<tr><th>Property</th><th>Value</th></tr>
<tr><td>`bubbles`</td><td>false</td></tr>
<tr><td>`cancelable`</td><td>false</td></tr>
<tr><td>`charCode`</td><td>The character code value of the key pressed or released.</td></tr>
<tr><td>`keyCode`</td><td>The key code value of the key pressed or released.</td></tr>
<tr><td>`keyLocation`</td><td>The location of the key on the AWKeyboard.</td></tr>
<tr><td>`ctrlKey`</td><td>true if the Control key is active; false if it is inactive.</td></tr>
<tr><td>`altKey`</td><td>false, reserved</td></tr>
<tr><td>`shiftKey`</td><td>true if the Shift key is active; false if it is inactive.</td></tr>
<tr><td>`currentTarget`</td><td>The Object that defines the
  event listener that handles the event. For example, if you use
  `comp.addEventListener()` to register an event listener,
  comp is the value of the `currentTarget`. </td></tr>
<tr><td>`target`</td><td>The Object that dispatched the event;
  it is not always the Object listening for the event.
  Use the `currentTarget` property to always access the
  Object listening for the event.</td></tr>

@eventType focusKeyDown
