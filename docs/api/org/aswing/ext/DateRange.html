API documentation

class org.aswing.ext.DateRange

The definition of a date range. For a single day, you can set rangeStart and rangeEnd to be a same date, If rangeStart == null, means all date before rangeEnd(included) is in range,
If rangeEnd == null, means all date after rangeStart(included) is in range.
